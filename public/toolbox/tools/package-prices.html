🔧 Site-Specific & Installation Terms (Add-ons)

If a customer's site requires extra materials, labor, or expanded battery backup, the actual costs listed below will be added to the project cost.

1
Battery Storage Expansion (Add-ons)

Standard hybrid packages include a base battery capacity (e.g., 5.12 kWh or 10.24 kWh). If a customer requests extended backup time, the actual wholesale cost of extra batteries, battery racks, and heavy-duty DC interconnect cables will be added directly to the project cost.

Example: Upgrading a 5-kWp system from 5.12 kWh to 10.24 kWh increases your project cost. Charge the customer accordingly to keep profit intact.
2
Extra Structure Height

Standard installation height is 5ft on the South side and 7ft on the North side. Any elevation beyond this adds ₹1,000 per extra foot to the project cost.

3
Extended Wiring (AC + DC)

Standard installations include up to 100 meters of total wiring. Any extra wire used adds ₹5,000 per 100 meters (calculated at ₹50 per meter) to the project cost.

Note: Extra heavy-duty battery cables beyond standard inverter-to-battery placement will be billed at actuals.
4
Concrete Footings

Every concrete footing (civil block) cast to secure structure legs to the roof adds ₹1,000 to the project cost.

5
On-Site Welding

Standard structures are bolted. If the site requires any custom on-site welding (e.g., attaching to existing iron fixtures), a flat fee of ₹2,000 is added to the project cost.

6
Fully Custom Structures

For highly specialized structures (heavy-steel carports, pergolas, elevated sheds, etc.), the exact actual cost of that custom fabrication is added to the project cost. SolarFlow will provide a custom quote for this specific fabrication upon site review.
